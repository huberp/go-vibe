## Documentation Guidelines

When adding new documentation:
1. Place it in the appropriate topic folder
2. Use clear, descriptive filenames (preferably uppercase for summaries/guides)
3. Update this README.md with a link to your new document
4. Update relevant references in other documentation files

## Important Notes

- **docs.go**: This file is auto-generated by Swagger and must remain in the root docs/ directory. It should never be moved or manually edited.
- **Swagger Files**: The swagger.json and swagger.yaml files are auto-generated but kept for reference and external tooling.
